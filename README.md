# وب‌سایت سفارش کیک و شیرینی


## تکنولوژی‌ها

- HTML5
- CSS3
- JavaScript
- Bootstrap 5 RTL

## صفحات

- `index.html` صفحه اصلی
- `products.html` صفحه محصولات
- `product-details.html` صفحه جزئیات محصول
- `order.html` صفحه ثبت سفارش
- `about.html` صفحه درباره ما
- `contact.html` صفحه تماس با ما

## امکانات

- طراحی فارسی و راست‌چین RTL
- طراحی Responsive برای موبایل، تبلت و دسکتاپ
- منوی موبایل Bootstrap
- جستجوی محصولات
- فیلتر محصولات بر اساس دسته‌بندی
- صفحه جزئیات محصول با Query String
- فرم ثبت سفارش با اعتبارسنجی JavaScript
- فرم تماس با ما با اعتبارسنجی JavaScript
- تصاویر SVG داخلی بدون نیاز به فایل تصویری خارجی
